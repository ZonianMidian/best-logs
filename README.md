# Best Logs by [Zonian](https://www.twitch.tv/ZonianMidian)

## What is this?

This service provides you with information about logs for a user on a Twitch channel

## Issues/Suggestions

You can create a [GitHub Issue](https://github.com/cqttv/best-logs/issues)

## Installation

- `git clone https://github.com/cqttv/best-logs.git`
- `cd best-logs`
- `npm install`
- `npm run build`
- `npm run start`
